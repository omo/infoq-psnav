
= InfoQ PSNAV =

This Greasemonkey userscript inserts slide navigaton buttons to 
InfoQ presentation video viewer pages (i.e. pages under http://www.infoq.com/presentations/)

Buttons include:
- go previous or next slides
  - shortcut key  ("p"-rev or "n"-ext) is available when mouse focus is on the guide.
- change slide side (zoomin, zoomout)
- show/hide all slides

The script also support slide file preloading,
which will be useful to skim inside slides before watching the videos.
Preloading will start on your first slide navigation.
